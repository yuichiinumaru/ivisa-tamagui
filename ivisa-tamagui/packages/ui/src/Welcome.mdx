# Bem-vindo ao Design System @ivisa/ui

Este é o catálogo de componentes, átomos, moléculas e organismos da nossa interface.

## Estrutura

- **Átomos**: Componentes básicos e indivisíveis (ex: Botões, Badges, Tipografia).
- **Moléculas**: Combinações de átomos (ex: Inputs com labels, Checkboxes com texto).
- **Organismos**: Estruturas complexas formadas por moléculas e átomos (ex: Tabelas, Dashboards, Navegação).

## Fontes

A fonte utilizada neste projeto é a **Cera Pro**.
Ela está sendo servida localmente para garantir consistência em todos os dispositivos.
